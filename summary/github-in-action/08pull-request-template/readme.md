### PR 템플릿 설정

PR을 만들 때 템플릿이 필요하다면 `pull_reqeust_template.md` 파일을 만들어 적용하는 것을 생각해볼 수 있습니다.
- public 디렉터리에 적용하고자 한다면 docs 디렉터리 하위에 파일을 저장한다.
- private 디렉터리에 적용하고자 한다면 .github 디렉터리 하위에 파일을 저장한다.
