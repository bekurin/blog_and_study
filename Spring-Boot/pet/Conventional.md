|Type|Description|
|-|-|
|Fix|버그를 고친 경우|
|Feat|새로운 기능을 추가한 경우|
|!BREAKING CHANGE| 코드에 큰 변화가 있는 경우|
|Refactor|리펙토링을 진행한 경우|
|Docs|문서를 추가 또는 수정한 경우|
|Test|테스트 추가, 테스트 리펙토링을 진행한 경우|
|Chore|짜잘한 코드 수정|
|Remove|파일을 삭제하는 경우|
|Rename|파일 또는 폴더명을 수정한 경우|

<br />

```
// Example
Feat: "추가 GET API 함수"
Test: "수정 POST Mapping 정보"
Rename: "고침 APIController -> ApiController"
```
